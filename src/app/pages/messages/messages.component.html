Messages

Message boxes and toast notifications for displaying important information

Toast Notifications

Floating notifications that appear temporarily

Inline Messages

Simple list-style messages with left border accent

info
This is an informational message
check_circle
Operation completed successfully
warning
Please review this warning
error
An error occurred

Message Boxes

Full message boxes with background and detailed content

info
Information
This is an informational message to provide helpful context to the user.
check_circle
Success
Your form has been submitted successfully. You will receive a confirmation email shortly.
warning
Warning
Please review your information carefully before proceeding. Some fields may need attention.
error
Error
Unable to process your request. Please check the form for errors and try again.

Usage Examples

Inline Message (HTML)

<div class="ft-msg ft-info">
  <span class="material-icons">info</span>
  <div class="ft-msg-text">Your message text here</div>
</div>

Message Box (HTML)

<div class="ft-msg-box ft-info">
  <span class="material-icons">info</span>
  <div class="ft-msg-content">
    <div class="ft-msg-title">Title (optional)</div>
    <div class="ft-msg-text">Your message text here</div>
  </div>
</div>

Toast Service (TypeScript)

import {{ '{' }} FtMessageService {{ '}' }} from '@feather/theme';

constructor(private messageService: FtMessageService) {{ '{' }}{{ '}' }}

// Simple toast
this.messageService.success('Operation completed!');

// Toast with title
this.messageService.info('New message', 'Notification');

// Toast with buttons
this.messageService.show({{ '{' }}
  type: 'warning',
  title: 'Confirm Action',
  message: 'Are you sure you want to continue?',
  buttons: [
    {{ '{' }} text: 'Yes', action: () => this.confirm() {{ '}' }},
    {{ '{' }} text: 'No', action: () => this.cancel() {{ '}' }}
  ]
{{ '}' }});
